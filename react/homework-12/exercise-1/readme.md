### Завдання - Багатомовний сайт за допомогою React Context

У папці `my-app` знаходиться простий додаток. В кількох компонентах є файли
перекладу `locale.json`. Зробіть вибір мови у компоненті `LanguageSwitcher` 
і відображення відповідного перекладу у компонентах, а також збереження 
вибранної мови в localStorage.
