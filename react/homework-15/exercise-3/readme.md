## Завдання 3 - Асинхроний Redux

### Частина 1 - список справ
Створіть за допомогою Redux toolkit `createSlice` список справ (todos), який працює з бекендом. Вимоги:
- створіть на `mockapi.io` бекенд для отримання, додавання, редагування списка справ;
- todo має містити назву, категорію (покупки, хатні справи, робота, інше) та пріорітетність (true або false);
- має бути 2 сторінки - на першій форма додавання todo та список, на другій - лише приорітені справи;
- в списку має бути кнопка видалення та редагування todo;
- список справ зберігайте в Redux і всі асинхроні операції робіть також через нього;

Для створення сторінок використайте React router dom у зручному для вас вигляді.

### Частина 2 - кошик
Створіть за допомогою Redux toolkit `createSlice` кошик (cart) и вибраним (favorites). Вимоги:
- створіть на `mockapi.io` бекенд для `products`, `cart` та `favorites`;
- створіть сторінку зі списком товарів (products). Товари можно створити списком і зберігати в окремому файлі;
- на картці товара має бути дві кнопки - Додати в кошик і Додати в обране;
- кошик має зберігатися в Redux  і всі асинхроні операції робіть також через нього;;
- вибране також має зберігатися в Redux і всі асинхроні операції робіть також через нього;
- створіть для кошика и вибране окремі сторінки;

Для створення сторінок використайте Tanstack Router.
