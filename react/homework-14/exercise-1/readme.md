### Завдання - React router dom 7

Створіть сайт із верхнім меню, у якому будуть такі посилання:
- Home (адреса="/");
- Products (адреса="/products");
- Products from partners (адреса="/products-partners");
- Contacts (адреса="/contacts");

І з відповідними компонентами-сторінками. Зробіть так, щоб при натисканні на посилання завантажувалася саме відповідна сторінка (сторінки можна зробити просто як <h2>Products</h2> тощо), а адресний рядок відповідно змінювався.
Зробіть це двома способами: через `Routes` та `Route` та `RouterProvider`
