### Завдання

Створіть форму реєстрації користувача з такими полями:
- Email;
- Ім’я;
- Пароль;
- тип користувача (user, manager, admin);

Після відправлення форми поряд із нею в окремому компоненті UserInfo виводьте отримані у формі дані у вигляді списку:
- Email користувача;
- Ім’я користувача;
- Тип. 

Вимоги до коду:
1. Елементи форми - текстові поля, чекбокс, випадаючий список - винесіть в окремі компоненти.
2. Налаштування полів винесіть в окремий файл в середині компоенту.
3. Використайте кастомний хук useForm.
